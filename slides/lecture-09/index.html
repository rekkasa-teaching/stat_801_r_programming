Στόχος του σημερινού μαθήματος

Μέχρι τώρα μάθαμε:

  1. Να επεξεργαζόμαστε δεδομένα με το dplyr.
  2. Να δημιουργούμε βασικά γραφήματα με το ggplot2 (geom_point, geom_histogram, κ.λπ.).


Σήμερα θα αφιερώσουμε το μάθημα στην αισθητική προσαρμογή (theming) των γραφημάτων.

Προετοιμασία: Φόρτωση και Βασικά Γραφήματα

library(dplyr)
library(ggplot2)

Δημιουργία των Γραφημάτων Βάσης (Base Plots)

Αντιγράψτε τον παρακάτω κώδικα

# p1: Ένα διάγραμμα διασποράς (Scatter plot)
p1 <- storms |> 
  filter(year == 2020) |> 
  ggplot(aes(x = wind, y = pressure, color = status)) +
  geom_point(size = 3, alpha = 0.7) +
  labs(title = "Ατμοσφαιρική Πίεση και Ταχύτητα Ανέμου (2020)",
       x = "Ταχύτητα Ανέμου (knots)", y = "Ατμοσφαιρική Πίεση (mb)",
       color = "Κατάσταση")

# p2: Ένα θηκόγραμμα (Boxplot)
p2 <- storms |> 
  filter(year %in% c(2000, 2010, 2020), status == "hurricane") |> 
  ggplot(aes(x = factor(year), y = wind, fill = factor(year))) +
  geom_boxplot() +
  labs(title = "Ταχύτητα Ανέμου Τυφώνων ανά Δεκαετία",
       x = "Έτος", y = "Ταχύτητα (knots)", fill = "Έτος")

Ενσωματωμένα Θέματα

Τα έτοιμα θέματα του ggplot2

Το ggplot2 έχει έτοιμα θέματα που αλλάζουν ριζικά την εμφάνιση με μία μόνο συνάρτηση. Το προεπιλεγμένο είναι το γκρι φόντο (theme_gray()).

Δοκιμάστε να τρέξετε:

p1 + theme_minimal() # Λευκό φόντο, χωρίς περίγραμμα
p1 + theme_bw()      # Κλασικό ασπρόμαυρο
p1 + theme_classic() # Εντελώς καθαρό (χωρίς πλέγμα, μόνο άξονες X/Y)
p1 + theme_light()   # Ανοιχτόχρωμο

Άσκηση

  1. Εμφανίστε το γράφημα p2 (το boxplot) στο RStudio.
  2. Προσθέστε σε αυτό το theme_minimal().
  3. Τι παρατηρείτε στο φόντο;
  4. Τώρα δοκιμάστε να εφαρμόσετε το theme_classic(). Πώς αλλάζει η απεικόνιση του πλέγματος (grid lines);

Η συνάρτηση theme()

Ρυθμός Καθηγητή (0:45) Εδώ ξεκινάει η πραγματική “δύναμη” του theming.

Πώς λειτουργεί η συνάρτηση theme()

  • Χρησιμοποιούμε τη συνάρτηση theme() η οποία δέχεται δεκάδες παραμέτρους (arguments).

  • Η λογική είναι πάντα: theme(όνομα_στοιχείου = συνάρτηση_στοιχείου())

  • Αν το στοιχείο είναι κείμενο, χρησιμοποιούμε τη συνάρτηση element_text().

Προσαρμογή κειμένων με element_text()

Ας αλλάξουμε την εμφάνιση του τίτλου (plot.title) και του άξονα X (axis.title.x).

p1 + 
  theme(
    # Αλλάζουμε τον κεντρικό τίτλο
    plot.title = element_text(
      color = "darkred",   # Χρώμα
      size = 18,           # Μέγεθος
      face = "bold",       # bold, italic, bold.italic
      hjust = 0.5          # Στοίχιση στο κέντρο (0 = αριστερά, 1 = δεξιά)
    ),
    
    # Αλλάζουμε τον τίτλο του άξονα X
    axis.title.x = element_text(
      face = "italic", 
      color = "blue"
    )
  )

Περιστροφή κειμένων

Το στοιχείο για το κείμενο πάνω στον άξονα λέγεται axis.text.x.

p2 + 
  theme(
    axis.text.x = element_text(angle = 45, hjust = 1)
  )


Το hjust = 1 βοηθάει ώστε το περιστραμμένο κείμενο να στοιχιστεί σωστά κάτω από τον άξονα.

Χρησιμοποιώντας το γράφημα p2:

  1. Βάλτε κεντρική στοίχιση στον τίτλο του γραφήματος (hjust = 0.5), κάντε τον έντονο (bold) και με μέγεθος 16.

Προσαρμογή υπομνήματος (Legend)

Η θέση του υπομνήματος

  • Το πιο συχνό ζητούμενο είναι η μετακίνηση του υπομνήματος (legend)
  • Εδώ δεν χρειαζόμαστε ειδική συνάρτηση, αρκεί ένα απλό κείμενο.
# Μεταφορά του υπομνήματος στο κάτω μέρος
p1 + theme(legend.position = "bottom")

# Μεταφορά στο επάνω μέρος
p1 + theme(legend.position = "top")

# Πλήρης αφαίρεση του υπομνήματος
p1 + theme(legend.position = "none")

Προσαρμογή εμφάνισης Υπομνήματος

Εφόσον το υπόμνημα έχει τίτλο (κείμενο) και φόντο (κουτί), μπορούμε να τα αλλάξουμε. Για τα κουτιά/επιφάνειες χρησιμοποιούμε τη συνάρτηση element_rect().

p1 + theme(
  # Αλλάζουμε το φόντο του υπομνήματος
  legend.background = element_rect(fill = "lightyellow", color = "black"),
  
  # Αλλάζουμε το κείμενο του τίτλου του υπομνήματος
  legend.title = element_text(face = "bold", color = "red")
)

Άσκηση

Πάρτε το γράφημα p1.

  1. Μετακινήστε το υπόμνημα στο κάτω μέρος του γραφήματος ("bottom").
  2. Αφαιρέστε τον τίτλο του υπομνήματος (“Κατάσταση”) χρησιμοποιώντας legend.title = element_blank(). Η συνάρτηση element_blank() εξαφανίζει οποιοδήποτε στοιχείο της αναθέσουμε!

Ρυθμός Καθηγητή (1:50 - 2:05) Εδώ εισάγουμε ομαλά το element_blank(). Εξηγήστε ότι είναι ο καλύτερος φίλος τους όταν θέλουν να ξεφορτωθούν κάτι άσχημο από το γράφημα.

Φόντο, γραμμές και element_blank()

Αλλαγή του background

Το φόντο χωρίζεται σε δύο μέρη: - panel.background: Ο χώρος που “κάθονται” τα δεδομένα. - plot.background: Το συνολικό χαρτί/καμβάς γύρω από το γράφημα.

p1 + 
  theme(
    panel.background = element_rect(fill = "lightblue"),
    plot.background = element_rect(fill = "gray90")
  )

Αλλαγή γραμμών: element_line()

  • Για να αλλάξουμε τις γραμμές χρησιμοποιούμε τη συνάρτηση element_line().

  • Οι γραμμές χωρίζονται σε κύριες (major) και δευτερεύουσες (minor).

p1 + 
  theme(
    panel.grid.major = element_line(color = "gray50", linetype = "dashed"),
    panel.grid.minor = element_line(color = "lightpink", size = 0.5)
  )


Τα είδη linetype είναι: “solid”, “dashed”, “dotted”, “dotdash” κ.α.

Καθαρίζοντας το γράφημα με element_blank()

Όπως είδαμε, το element_blank() εξαφανίζει στοιχεία. Είναι ιδανικό για να αφαιρούμε άχρηστες γραμμές ή πληροφορίες που φορτώνουν την εικόνα.

p2 + 
  theme_bw() + 
  theme(
    # Εξαφανίζουμε τις δευτερεύουσες γραμμές
    panel.grid.minor = element_blank(),
    
    # Εξαφανίζουμε τις κάθετες κύριες γραμμές (του άξονα Χ)
    panel.grid.major.x = element_blank()
  )

Άσκηση

Ξεκινήστε με το γράφημα p2.

Προσθέστε επίπεδα (+) ώστε να:

  1. Εφαρμόσετε το έτοιμο theme_classic().
  2. Μετακινήσετε το υπόμνημα στο επάνω μέρος (“top”).
  3. Προσθέσετε μόνο οριζόντιες γραμμές πλέγματος (panel.grid.major.y) οι οποίες θα είναι διακεκομμένες (linetype = "dashed") και με χρώμα ανοιχτό γκρι ("gray80").
