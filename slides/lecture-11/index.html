Μέρος 1ο: Εξερεύνηση και καθαρισμός

Προετοιμασία: Φόρτωση βιβλιοθηκών

Ξεκινάμε φορτώνοντας τα δύο βασικά μας πακέτα.

library(dplyr)
library(ggplot2)


Δεν χρειάζεται να φορτώσουμε δεδομένα από αρχείο! Το dataset starwars είναι ήδη ενσωματωμένο στο dplyr.

Το dataset starwars

Περιέχει πληροφορίες για τους χαρακτήρες της σειράς ταινιών Star Wars.

starwars


glimpse(starwars)

Άσκηση

Χρησιμοποιώντας το starwars:

  1. Πόσοι χαρακτήρες υπάρχουν συνολικά;
  2. Πόσα διαφορετικά είδη (species) εμφανίζονται; (Συμβουλή: distinct() + nrow(), ή n_distinct()).
  3. Πόσοι χαρακτήρες έχουν NA στη στήλη mass; (Συμβουλή: filter(is.na(mass)) και μετά nrow()).
  4. Ποιοι είναι οι 5 ψηλότεροι χαρακτήρες; Εμφανίστε τις στήλες name, height, species.

Πόσα NA έχουμε σε κάθε στήλη;

starwars |>
  summarise(
    n_height     = sum(is.na(height)),
    n_mass       = sum(is.na(mass)),
    n_birth_year = sum(is.na(birth_year)),
    n_homeworld  = sum(is.na(homeworld)),
    n_species    = sum(is.na(species))
  )

Παρατηρούμε ότι η στήλη birth_year έχει πολλά NA. Πρέπει να αποφασίσουμε πώς θα τα χειριστούμε όταν την χρησιμοποιούμε.

Καθαρισμός: Φιλτράρισμα και μετασχηματισμός

Φτιάχνουμε ένα καθαρό dataset, το sw_clean:

sw_clean <- starwars |>
  filter(!is.na(height), !is.na(mass)) |>      # κρατάμε όσους έχουν height & mass
  filter(mass < 500) |>                        # αφαιρούμε ακραίες τιμές (π.χ. Jabba)
  mutate(
    height_m = height / 100,                   # μετατροπή σε μέτρα
    bmi      = mass / (height_m ^ 2)           # δείκτης μάζας σώματος
  ) |>
  select(name, species, homeworld, sex, height, mass, height_m, bmi)

glimpse(sw_clean)

Άσκηση

Ξεκινώντας από το αρχικό starwars, δημιουργήστε ένα νέο tibble με όνομα sw_adults που:

  1. Κρατάει μόνο όσους έχουν γνωστό birth_year (όχι NA).
  2. Δημιουργεί μια νέα στήλη age_at_yavin που δείχνει την ηλικία του χαρακτήρα τη χρονιά της Μάχης του Yavin (BBY = 0). Συμβουλή: το birth_year είναι χρόνια πριν τη Μάχη του Yavin, οπότε αυτό είναι και η ηλικία τους.
  3. Φιλτράρει για ενήλικες (ηλικία ≥ 18).
  4. Επιλέγει τις στήλες: name, species, homeworld, age_at_yavin.
  5. Ταξινομεί με βάση την ηλικία φθίνουσα (από τον πιο γέρο στον νεότερο).

Μέρος 2ο: Βοηθητικοί πίνακες και Joins

Γιατί χρειαζόμαστε επιπλέον πίνακες;

Το starwars έχει πολλά διαφορετικά είδη (Human, Droid, Wookiee, …) και δεκάδες πλανήτες.


Συχνά θέλουμε να ομαδοποιήσουμε σε μεγαλύτερες κατηγορίες ή να εμπλουτίσουμε με πληροφορία που δεν υπάρχει στο αρχικό dataset:

  • Κατηγορία είδους (humanoid / droid / mammal / …)
  • Περιοχή του γαλαξία (Core / Mid Rim / Outer Rim) ανά πλανήτη
  • Κλίμα του πλανήτη


Αυτές τις πληροφορίες θα τις φτιάξουμε μόνοι μας με τη συνάρτηση tribble().

Η συνάρτηση tribble()

Η tribble() φτιάχνει έναν μικρό tibble γράφοντας τις γραμμές όπως σε ένα πίνακα:

example <- tribble(
  ~name,    ~grade,
  "Maria",  9.5,
  "John", 7.0,
  "George",  8.5
)

example


Τα ονόματα των στηλών μπαίνουν με ~ μπροστά. Κάθε επόμενη γραμμή είναι μια καινούργια παρατήρηση.

Ομαδοποιούμε τα είδη σε κατηγορίες:

Πρώτο join: εμπλουτίζουμε το sw_clean

Θέλουμε να προσθέσουμε τη στήλη classification δίπλα σε κάθε χαρακτήρα του sw_clean.

sw_class <- sw_clean |>
  left_join(species_class, join_by(species))

glimpse(sw_class)


Χρησιμοποιήσαμε left_join ώστε να κρατήσουμε όλους τους χαρακτήρες, ακόμα κι όσοι έχουν είδος για το οποίο δεν φτιάξαμε classification (θα πάρουν NA).

Άσκηση

Χρησιμοποιώντας το sw_class:

  1. Αφαιρέστε όσους χαρακτήρες έχουν NA στο classification.
  2. Ομαδοποιήστε ανά classification.
  3. Υπολογίστε:
    • Το πλήθος χαρακτήρων (n_characters).
    • Το μέσο ύψος σε μέτρα (avg_height_m).
    • Το μέσο BMI (avg_bmi).
  4. Ταξινομήστε με βάση τον μέσο BMI σε φθίνουσα σειρά.

Ρυθμός Καθηγητή (1:10 - 1:30) Πιθανώς θα έχουν αξιοπρόσεκτα αποτελέσματα: τα Mammals (Wookiees, Ewoks) έχουν υψηλό BMI. Σχολιάστε ότι το na.rm = TRUE εδώ ΔΕΝ είναι απαραίτητο αφού στο sw_clean έχουμε ήδη πετάξει τα NA από height/mass. Σημείωση: στο sw_class η στήλη species εξακολουθεί να υπάρχει — απλά έχει προστεθεί και η classification.

Δεύτερο join: ενώνουμε δύο βοηθητικούς πίνακες

sw_full <- sw_clean |>
  left_join(species_class,  join_by(species)) |>
  left_join(homeworld_info, join_by(homeworld))

glimpse(sw_full)


Τώρα κάθε χαρακτήρας έχει: αρχικά δεδομένα + classification + galaxy_region + climate.

Άσκηση

Χρησιμοποιώντας το sw_full:

  1. Αφαιρέστε όσους χαρακτήρες έχουν NA στο galaxy_region.
  2. Ομαδοποιήστε ανά galaxy_region και classification.
  3. Υπολογίστε:
    • Το πλήθος χαρακτήρων (n).
    • Το μέσο ύψος σε μέτρα (avg_height_m).
  4. Ταξινομήστε ανά galaxy_region και κατόπιν φθίνον n.

Μέρος 3ο: Σύνθεση και Γραφήματα

Από τους πίνακες στα γραφήματα

Έχουμε ήδη φτιάξει το sw_full που περιέχει όλη τη συνδυασμένη πληροφορία.


Τώρα θα δούμε τρία είδη γραφημάτων που απαντούν σε τρία διαφορετικά ερωτήματα:

  1. Bar chart: Πόσοι χαρακτήρες ανά κατηγορία είδους;
  2. Boxplot: Πώς κατανέμεται το ύψος ανά κατηγορία;
  3. Scatter plot με facets: Πώς σχετίζεται ύψος ↔︎ μάζα ανά περιοχή γαλαξία;

Γράφημα 1ο: Bar chart πλήθους

sw_full |>
  filter(!is.na(classification)) |>
  count(classification) |>
  ggplot(aes(x = reorder(classification, n), y = n, fill = classification)) +
  geom_col(show.legend = FALSE) +
  coord_flip() +
  labs(
    title = "Πλήθος χαρακτήρων ανά κατηγορία είδους",
    x = NULL,
    y = "Αριθμός χαρακτήρων"
  ) +
  theme_minimal()

Γράφημα 2ο: Boxplot υψών ανά classification

sw_full |>
  filter(!is.na(classification)) |>
  ggplot(aes(x = classification, y = height_m, fill = classification)) +
  geom_boxplot(show.legend = FALSE) +
  labs(
    title = "Κατανομή υψών ανά κατηγορία είδους",
    x = "Κατηγορία",
    y = "Ύψος (m)"
  ) +
  theme_classic() +
  theme(
    plot.title  = element_text(face = "bold", hjust = 0.5),
    axis.text.x = element_text(angle = 30, hjust = 1)
  )

Άσκηση

Χρησιμοποιώντας το sw_full:

  1. Αφαιρέστε όσους έχουν NA στο galaxy_region.
  2. Φτιάξτε ένα scatter plot με:
    • Άξονας X: height_m
    • Άξονας Y: mass
    • Χρώμα: classification
  3. Προσθέστε geom_point(size = 3, alpha = 0.7).
  4. Κάντε facet_wrap(~ galaxy_region).
  5. Προσθέστε κατάλληλους τίτλους με τη labs().
  6. Εφαρμόστε theme_minimal() και βάλτε το υπόμνημα στο κάτω μέρος.

Ρυθμός Καθηγητή (2:25 - 2:45) Αυτή η άσκηση συνδυάζει join + filter + ggplot + theme + facet. Είναι σχεδόν αναπαράσταση όλης της ύλης. Αν κάποιοι ψάχνουν για περισσότερη πρόκληση, ζητήστε τους να αλλάξουν τους labels του υπομνήματος (labs(color = "Κατηγορία")).
