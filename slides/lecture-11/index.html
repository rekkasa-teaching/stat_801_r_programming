Συμπλήρωση Κενών

Συμπλήρωση κενών

Συμπληρώστε τα κενά (____) ώστε η ροή να: κρατά όσους έχουν γνωστό height & mass, αφαιρεί ακραίες τιμές, ομαδοποιεί ανά είδος, και υπολογίζει πλήθος + μέσο ύψος, ταξινομημένα φθίνουσα ως προς το πλήθος.

sw_summary <- starwars |>
  ________(!is.na(height), !is.na(mass)) |>    # (1) κρατάμε γνωστά height & mass
  ________(mass < 500) |>                      # (2) αφαιρούμε ακραίες τιμές
  mutate(height_m = ______ / ______) |>        # (3) μετατροπή από εκατοστά σε μέτρα
  ________(species) |>                         # (4) ομαδοποίηση ανά είδος
  summarise(
    n          = ____(),                       # (5) πλήθος χαρακτήρων
    avg_height = mean(________),               # (6) μέσο ύψος σε μέτρα
    .groups    = "drop"
  ) |>
  ________(desc(n))                            # (7) ταξινόμηση φθίνουσα

sw_summary

Σωστό / Λάθος

Σωστό ή Λάθος;

Σημειώστε Σ ή Λ για κάθε πρόταση.

  1. Το filter() επιλέγει στήλες από έναν πίνακα.
  2. Το mutate() δημιουργεί ή τροποποιεί στήλες.
  3. Μετά από group_by() πρέπει πάντα να ακολουθεί summarise().
  4. Το arrange(desc(x)) ταξινομεί αύξουσα.
  5. Ο τελεστής |> περνά το αριστερό αποτέλεσμα ως πρώτο όρισμα της επόμενης συνάρτησης.
  6. Το left_join() μπορεί να εισάγει NA σε γραμμές που δεν βρήκαν ταίριασμα.
  7. Στο geom_boxplot() ο άξονας X πρέπει να είναι αριθμητικός.

Σωστό ή Λάθος; (συνέχεια)

  1. Το count(x) επιστρέφει μια στήλη πλήθους με όνομα n.
  2. Το select(-name) αφαιρεί τη στήλη name.
  3. Το inner_join() κρατά και γραμμές που δεν βρήκαν ταίριασμα.
  4. Στο aes() μπορούμε να αντιστοιχίσουμε ένα χρώμα σε μια μεταβλητή.
  5. Το mean(c(1, 2, NA), na.rm = TRUE) επιστρέφει NA.
  6. Το group_by() αλλάζει τον αριθμό των γραμμών του πίνακα.

Ολοκληρωμένη ερώτηση συμπλήρωσης

Capstone: συμπλήρωση κενών (1/2)

Συμπληρώστε τα κενά ώστε, ξεκινώντας από το starwars, να φτιαχτεί boxplot ύψους ανά κλίμα πλανήτη, αφού γίνουν δύο joins με τους βοηθητικούς πίνακες species_class και homeworld_info.

sw_full <- starwars |>
  filter(!is.na(height), !is.na(mass)) |>
  filter(mass ____ 500) |>                        # (1) αφαίρεση ακραίων
  mutate(height_m = height / 100) |>
  left_join(species_class,  join_by(________)) |>  # (2) κλειδί ένωσης
  left_join(homeworld_info, join_by(________))     # (3) κλειδί ένωσης

Capstone: συμπλήρωση κενών (2/2)

sw_full |>
  ________(!is.na(climate)) |>                    # (4) μόνο γνωστό κλίμα
  ggplot(aes(x = climate, y = height_m, fill = climate)) ____  # (5) σύνδεση layers
  ________() +                                     # (6) geom για boxplot
  labs(title = "Ύψος ανά κλίμα πλανήτη", x = "Κλίμα", y = "Ύψος (m)") +
  theme_minimal() +
  theme(legend.position = "____")                  # (7) απόκρυψη υπομνήματος
